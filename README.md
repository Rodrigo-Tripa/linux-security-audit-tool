# Linux Security Audit Tool

> ⚠️ Status: Under Development

A lightweight and modular Bash-based tool designed to audit the security posture of Linux systems. The tool aims to identify common misconfigurations and provide actionable security insights aligned with CIS Benchmarks and MITRE ATT&CK techniques.

## Features (Planned)

- User and group enumeration
- SUID/SGID and world-writable file detection
- SSH configuration auditing
- Network and running services analysis
- Basic system hardening checks
- Report generation

## Usage (Planned)

```bash
sudo ./linux-security-audit.sh
